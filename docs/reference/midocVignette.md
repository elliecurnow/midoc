# Run an interactive vignette for the midoc package

Runs an interactive version of the midoc vignette: Multiple Imputation
DOCtor (midoc). In the interactive version, you can apply midoc
functions in
[shiny-package](https://rdrr.io/pkg/shiny/man/shiny-package.html) apps
using your own DAG and data.

## Usage

``` r
midocVignette()
```

## Value

A browser-based, interactive version of the midoc vignette

## Examples

``` r
if (FALSE) { # interactive()
# Run the interactive vignette
midocVignette()
}
```
