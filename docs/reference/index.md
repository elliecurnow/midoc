# Package index

## Explore Data

Explore missing data patterns and mechanisms

- [`descMissData()`](https://elliecurnow.github.io/midoc/reference/descMissData.md)
  : Lists missing data patterns in the specified dataset
- [`exploreDAG()`](https://elliecurnow.github.io/midoc/reference/exploreDAG.md)
  : Compares data with proposed DAG

## Check Validity

Check whether complete records analysis and multiple imputation are
valid

- [`checkCRA()`](https://elliecurnow.github.io/midoc/reference/checkCRA.md)
  : Inspect complete records analysis model
- [`checkMI()`](https://elliecurnow.github.io/midoc/reference/checkMI.md)
  : Inspect multiple imputation model

## Implement MI

Implement a suitable multiple imputation strategy

- [`checkModSpec()`](https://elliecurnow.github.io/midoc/reference/checkModSpec.md)
  : Specify and inspect parametric model specification
- [`proposeMI()`](https://elliecurnow.github.io/midoc/reference/proposeMI.md)
  : Suggests multiple imputation options
- [`doMImice()`](https://elliecurnow.github.io/midoc/reference/doMImice.md)
  : Performs multiple imputation
- [`doMNARMImice()`](https://elliecurnow.github.io/midoc/reference/doMNARMImice.md)
  : Performs not-at-random multiple imputation
- [`doRefBasedMI()`](https://elliecurnow.github.io/midoc/reference/doRefBasedMI.md)
  : Performs reference-based multiple imputation

## Examples and Vignettes

Example datasets and interactive versions

- [`bmi`](https://elliecurnow.github.io/midoc/reference/bmi.md) : Child
  body mass index data
- [`qol`](https://elliecurnow.github.io/midoc/reference/qol.md) :
  Randomised controlled trial data
- [`adr`](https://elliecurnow.github.io/midoc/reference/adr.md) :
  Administrative data
- [`midocVignette()`](https://elliecurnow.github.io/midoc/reference/midocVignette.md)
  : Run an interactive vignette for the midoc package
- [`midocShinyApp()`](https://elliecurnow.github.io/midoc/reference/midocShinyApp.md)
  : Run a browser-based version of the midoc package
