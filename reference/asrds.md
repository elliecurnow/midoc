# Adapted Self-Report Delinquency Scale data

A simulated dataset

## Usage

``` r
asrds
```

## Format

### `asrds`

A data frame with 1000 rows and 5 columns:

- asrds17:

  Child's Adapted Self-Report Delinquency Scale (ASRDS) value at age 17
  years, as a percentage of the maximum possible value of the
  square-root of ASRDS

- log_income5:

  Logarithm of the family income per annum, in UK pounds, when the child
  is aged 5 years

- mated:

  Mother's educational level at time of pregnancy: post-16 years
  qualification or not

- asrds14:

  Child's ASRDS value at age 14 years, as a percentage of the maximum
  possible value of the square-root of ASRDS

- r_cra:

  Missingness indicator: complete record (case) or not
